# Cow Strike 1


Software that calculates milk production for each color of cow.

## Tech Stack

- __Backend__ : Node.js
- __Frontend__ : EJS, HTML, CSS
- __Web Service__ : Express.js
- __Database__ : MySQL
- __Tools__ : Postman
